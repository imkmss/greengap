## Green Gap Analysis

### 기준 기반 판단

* 정책 기준(1인당 6m²/인, 800m 접근권) 대비 달성 수준 비교
* 지역별 녹지 부족 심각도 정량화

### 취약 지역 라벨링

* 부족 여부 라벨링 → 이후 ML 모델 학습용 데이터셋으로 활용
* 형평성·환경 개선 효과·접근성·실행 가능성을 종합 점수화하여 시급 우선순위 구역 등급 산정
