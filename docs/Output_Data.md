# Output Data

Green Gap Analysis 결과를 공간 데이터와 통계 데이터 두 형식으로 출력하는 단계.

---

## GeoJSON (지도)

* 공간 분석 결과를 지도 형태로 저장
* Leaflet 기반 프론트엔드에서 인터랙티브 시각화에 활용
* 지역 선택 시 녹지율, 1인당 녹지 면적, 공원 접근성 실시간 조회 가능

## CSV (통계)

* 행정동 단위 집계 지표를 테이블 형태로 저장
* 녹지 부족 등급, 우선순위 순위 등 포함
* 예산 분배·사업 계획 수립의 데이터 기반 근거로 활용
