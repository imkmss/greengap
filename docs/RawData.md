# Raw Data

파이프라인의 첫 단계로, 분석에 필요한 원시 데이터를 수집한다.

---

## 토지피복도

* 출처: 환경부 / 환경공간정보서비스(EGIS)
* 대상: 서울시 전체
* 형식: Shapefile (.shp)
* 주요 필드:

  | 필드 | 설명 |
  |------|------|
  | `L1_CODE` / `L1_NAME` | 대분류 코드 / 이름 |
  | `L2_CODE` / `L2_NAME` | 중분류 코드 / 이름 |
  | `L3_CODE` / `L3_NAME` | 소분류 코드 / 이름 |
